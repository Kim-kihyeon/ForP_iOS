# ForP — AI 데이트 코스 앱

> **For Partners** | P성향 커플을 위한 맞춤 데이트 코스 생성 서비스

---

## 앱 소개

ForP는 데이트 코스 선정을 고민하는 커플을 위한 iOS 앱이다.
"어디 가?" "뭐 먹지?" 를 반복하는 대신, AI가 두 사람의 취향·날씨·지역·테마를 종합해 최적의 데이트 코스를 자동으로 생성해준다.

특히 즉흥적이고 계획 세우는 걸 부담스러워하는 **P성향(Perceiving)** 사람들을 위해 설계됐다.

## 핵심 기능

### 1. AI 코스 생성
- 입력값: 지역, 날짜, 장소 수(2~6곳), 테마, 요청사항
- **실제 날씨 반영**: OpenWeather API로 오늘 날씨는 현재 기상, 최대 4일 이내는 예보, 그 이후는 계절 추정
- **지오코딩**: Kakao 장소 검색으로 한국 지명 정확하게 좌표 변환
- GPT가 장소 추천 이유, 추천 메뉴, 코스 이유, 의상 제안까지 생성
- 선택되지 않은 **후보 장소**도 함께 제공 → 저장 후에도 유지

### 2. 취향 학습
- 코스 생성 시 선택한 테마를 자동으로 학습
- 다음 코스 생성 시 이전에 선택한 테마 자동 선택
- AI 프롬프트에 선호 테마 반영 → 갈수록 취향에 맞는 코스 생성

### 3. 코스 결과 뷰
- **지도**: 장소들이 번호 마커로 표시되고 동선이 선으로 연결
- **타임라인**: 장소별 추천 이유, 카테고리, 주소, 추천 메뉴 표시
- **찜**: 각 장소를 개별적으로 찜 목록에 저장
- **지도 연동**: 카카오맵으로 바로 열기
- **순서 편집**: 드래그로 장소 순서 변경, 장소 삭제
- **후보 장소**: AI가 선택하지 않은 대안 장소들 확인 및 코스에 추가

### 4. 데이트 진행 모드
- "데이트 시작" 버튼으로 실시간 진행 모드 전환
- 각 장소 방문 체크 → 완료 시 진행률 표시
- 모든 장소 방문 완료 시 별점 + 한 줄 후기 입력 시트
- 후기는 코스에 저장되어 나중에도 확인 가능

### 5. 코스 관리
- **최근 코스**: 홈 화면에서 생성한 코스 목록 확인
- **즐겨찾기**: 하트 버튼으로 즐겨찾기 등록 → 홈 상단 가로 스크롤 카드로 노출
- **평점/후기**: 별 1~5개 + 텍스트 후기
- **공유**: 코스 내용을 텍스트로 공유

### 6. 파트너 프로필
- 파트너 닉네임, 선호/비선호 카테고리, 메모 등록
- 카테고리: 카페, 브런치, 음식점, 술/바, 영화, 공원, 전시, 문화, 쇼핑, 액티비티, 드라이브, 노래방, 스포츠, 야경, 힐링
- 파트너 취향이 AI 코스 생성에 반영됨

### 7. 내 프로필
- 닉네임, 자주 가는 지역 설정
- 선호/비선호 카테고리 설정
- 중복 선택 방지 (선호에 있으면 비선호 선택 불가)

### 8. 기념일 관리
- 커플 기념일 등록 (이름 + 날짜)
- D-Day 카운트다운 자동 계산
- N주년 표시
- 홈 화면에 가장 가까운 기념일 카드로 표시

### 9. 찜 목록
- 코스 결과에서 찜한 장소 목록 관리
- 최대 20개
- 코스 생성 시 찜한 장소 최대 3개 포함 가능 (지역 다르면 AI가 유사 유형으로 대체)

### 10. 부가 기능
- **출발 시각 계산기**: 각 장소 도착 목표 시간 기반으로 역산
- **예산 계산기**: 장소별 예산 입력 및 합산
- **체크리스트**: 데이트 준비물 체크

## 홈 화면

- 현재 날씨 + 데이트 힌트 (ex. "야외 데이트 딱 좋은 날씨")
- 다가오는 기념일 카드
- 즐겨찾기 코스 가로 스크롤
- 최근 코스 목록 (날짜 배지, 평점 별 표시)
- 하단 "코스 만들기" 버튼

## 기술 스택

| 영역 | 기술 |
|------|------|
| UI 프레임워크 | SwiftUI |
| 아키텍처 | TCA (The Composable Architecture) |
| 백엔드/DB | Supabase (PostgreSQL + REST API) |
| AI | OpenAI GPT-4o |
| 날씨 | OpenWeather API |
| 지도/장소 | Kakao Local API, MapKit |
| 인증 | Apple Sign In, Kakao Login |
| 배포 | fastlane + GitHub Actions → TestFlight |

### 모듈 구조
```
ForP
├── Modules
│   ├── Core
│   │   └── SharedUI       # 공통 컴포넌트, Brand 컬러, Typography
│   ├── Domain             # 엔티티, UseCase, Repository 인터페이스
│   ├── Data               # Supabase/API 구현체
│   └── Features           # 화면별 Feature (TCA Reducer + View)
│       ├── Home
│       ├── CourseGenerate
│       ├── CourseResult
│       ├── Profile
│       ├── Partner
│       ├── Anniversary
│       ├── Wishlist
│       ├── Settings
│       ├── Login
│       └── Onboarding
```

## 화면 흐름

```
로그인 (Apple / 카카오)
    ↓
온보딩 (지역 · 카테고리 취향 · 분위기 테마)
    ↓
홈 화면
    ├── 코스 만들기 → 코스 생성 → 코스 결과 → 데이트 진행
    ├── 기존 코스 선택 → 코스 결과
    └── 설정 → 파트너 / 내 프로필 / 기념일 / 찜 목록
```

## 디자인 컨셉

- **브랜드 컬러**: 핑크(#FF2D55 계열) + 소프트핑크
- 커플 앱 특성상 따뜻하고 로맨틱한 톤
- 코스 생성 로딩: 풀스크린 블러 + 리플 링 애니메이션
- 카드 기반 레이아웃, 그림자와 그라디언트로 depth 표현
- 다크모드 지원

## 타겟 사용자

- 데이트 코스 선정을 매번 고민하는 커플
- 계획 세우기 귀찮은 P성향 사람
- 새로운 장소를 발견하고 싶은 커플
- 기념일/특별한 날에 완벽한 코스를 원하는 사람
